# 📚 سیستم مدیریت کتابخانه

یک برنامه دسکتاپ ساده برای مدیریت کتابخانه، ساخته‌شده با Python، Tkinter و SQLite.

---

## ✨ امکانات

**مدیریت کتاب‌ها**
- افزودن، ویرایش و حذف کتاب
- جستجو بر اساس عنوان
- نمایش اطلاعات کامل (عنوان، نویسنده، ژانر، سال، وضعیت)

**مدیریت کاربران** _(فقط برای ادمین)_
- ثبت‌نام و ورود با نام کاربری و رمز عبور
- افزودن، ویرایش و حذف کاربران
- سطح دسترسی دوگانه: کاربر عادی / ادمین

---

## 🛠 پیش‌نیازها

- Python 3.x
- کتابخانه‌های `tkinter` و `sqlite3` (از پیش در Python نصب هستند)

---

## 🚀 نحوه اجرا

```bash
git clone https://github.com/your-username/library-manager.git
cd library-manager
python library.py
```

دیتابیس (`library.db`) به صورت خودکار در همان پوشه ساخته می‌شود.

---

## 📁 ساختار پروژه

```
library-manager/
│
├── library.py      # فایل اصلی برنامه
├── library.db      # دیتابیس SQLite (پس از اجرا ساخته می‌شود)
└── README.md
```

---

## 🖼 نمای کلی برنامه

| صفحه | توضیح |
|------|-------|
| ورود | احراز هویت با نام کاربری و رمز عبور |
| پنجره اصلی | نمایش لیست کتاب‌ها با امکان مدیریت |
| پنل ادمین | مدیریت کاربران (فقط برای ادمین) |

## ⚠️ نکات امنیتی

> این پروژه برای اهداف آموزشی طراحی شده است. در محیط واقعی توصیه می‌شود:
> - رمزهای عبور به صورت hash ذخیره شوند (مثلاً با `bcrypt`)
